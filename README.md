# voyage-analytics-

Designed and built an end-to-end machine learning platform covering data analysis, model training, experiment tracking (MLflow), and production inference.
Exposed ML models through Flask REST APIs, containerized them with Docker, and orchestrated services locally using Kubernetes (Minikube).
Validated models via API testing and managed model versions and metrics using MLflow.
Delivered a user-friendly Streamlit web application integrating all models, deployed via GitHub for interactive analytics and predictions.
